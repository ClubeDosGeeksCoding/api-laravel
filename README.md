# api-laravel
API Example using Laravel

## Install
```shell 
git clone https://github.com/ClubeDosGeeksCoding/api-laravel.git
cd api-laravel
composer install
```

## Configure Virtual Hosts
```xml
<VirtualHost *:80>
    DocumentRoot "CAMINHO_PARA_PASTA_DA_API"
    ServerName laravel.exemple
</VirtualHost>
```
